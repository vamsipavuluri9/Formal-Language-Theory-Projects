Main Note ( If running on a unix machine):
------------------------------------------
While providing input files ( i.e., DFSM.txt file and input.txt files) please first convert those into unix readable standard format.

Use the command: dos2unix textfilename.txt 
(for every input text file first execute this command before writing ./a.out DFSM.txt input.txt or something like this)
